Switch Statements used to compare the values of a variable/expression with
multiple constant cases and executes the matching case.
it is an alternative to if-else-if ladder

keywords : switch(expression/variable)
            case value:
                statement
                break;
            case value:
                statement
                break;
            default ;
                statement

Data types not allowed : long, float, double, boolean ,object types
Data types allowed : byte,short,int,string,enum,char.
